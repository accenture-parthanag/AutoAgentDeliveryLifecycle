1. Summary

This TDD decomposes the approved Graphic Autonumber SDD into atomic, implementable engineering tasks on Microsoft Power Platform (Power Apps canvas + Power Automate + Dataverse + SharePoint + Outlook + Teams). Implementation proceeds in four phases: (1) tenant/environment + Dataverse schema + identity, (2) Power Automate flows (Autonumber + Status Change) and Outlook templates, (3) Power Apps canvas screens (Landing, Create Tracker, Open Tracker, View Tracker, New Item, Edit Item) wired to Dataverse and the flows, and (4) Teams tab packaging, RBAC hardening, observability and UAT. Each task traces to a named SDD component and (for UI work) an SDD screen id, and uses the autonumber format <OppID>.<YY>.<NNNN> defined by BT in Q1.


Build sequence


Phase 1 â€” Foundation: Power Platform environment, Dataverse solution & tables (GraphicTracker, GraphicRequest, OrderSequence), security roles, Entra ID/Teams identity wiring.


Phase 2 â€” Workflow Backbone: AutonumberFlow (atomic OrderSequence increment + control-number stamp), StatusChangeFlow (Updates Needed / Completed routing), Outlook email templates and read-only enforcement on GraphicControlNumber.

Pseudocode / implementation sketch

// C# Dataverse plugin (PreOperation Update on gt_graphicrequest)

public class LockControlNumberPlugin : IPlugin {

  public void Execute(IServiceProvider sp) {

    var ctx = (IPluginExecutionContext)sp.GetService(typeof(IPluginExecutionContext));

    if (!ctx.InputParameters.Contains("Target") || !(ctx.InputParameters["Target"] is Entity target)) return;

    if (!target.Attributes.Contains("gt_graphiccontrolnumber")) return;

    var pre = (Entity)ctx.PreEntityImages["PreImage"];

    var oldVal = pre.GetAttributeValue<string>("gt_graphiccontrolnumber");

    var newVal = target.GetAttributeValue<string>("gt_graphiccontrolnumber");

    if (!string.IsNullOrEmpty(oldVal) && oldVal != newVal)

      throw new InvalidPluginExecutionException("GraphicControlNumber is read-only after generation (BT Q2).");

  }

}

Description

Create the Power Automate cloud flow that triggers on gt_graphicrequest row create. It resolves the parent gt_graphictracker to read gt_opportunityid, computes sequenceKey = OppId + '.' + last-two-digits(year(utcNow)), upserts the gt_ordersequence row, increments gt_lastorder atomically using If-Match ETag with up to 5 retries (exponential jitter), formats <OppId>.<YY>.<NNNN>, and patches gt_graphiccontrolnumber on the originating row. On final failure logs to flow run history and surfaces a 'Control number pending' placeholder on the row (BT Q5 platform-outage mitigation).


Acceptance criteria


Trigger fires within 5s of a row create on gt_graphicrequest in DEV environment.


Two concurrent inserts for the same OppId+Year produce two distinct, consecutive control numbers (e.g., .0001 and .0002) â€” verified by parallel-load script.


Forced 412 from Dataverse on the OrderSequence update is retried up to 5x with jittered backoff; on success the final value is correct.


Generated value matches regex ^\d{6}\.\d{2}\.\d{4}$.


After successful stamp, the New-Graphics email is dispatched (handed off to task 8).


Pseudocode / implementation sketch

// Power Automate â€” AutonumberFlow

Trigger: 'When a row is added' (Dataverse) â€” Table: gt_graphicrequest

 

Init variables:

  varOppId   = triggerOutputs()?['body/_gt_trackerid_value@OData.Community.Display.V1.FormattedValue']

  // Lookup tracker to get OppId

  Tracker = Get a row by ID (gt_graphictracker, triggerOutputs()?['body/_gt_trackerid_value'])

  varOppId = Tracker['gt_opportunityid']

  varYY    = substring(string(utcNow('yy')), 0, 2)

  varKey   = concat(varOppId, '.', varYY)

  varRetry = 0

  varStamped = false

 

Do Until (varStamped or varRetry >= 5):

  Get sequence row (alternate key gt_sequencekey eq varKey)

    if Not Found:

      Add new row gt_ordersequence { gt_sequencekey: varKey, gt_opportunityid: varOppId, gt_year: varYY, gt_lastorder: 0 }

  Capture etag = body['@odata.etag']

  varNext = add(int(seqRow['gt_lastorder']), 1)

  Update row gt_ordersequence id=seqRow.id with body { gt_lastorder: varNext } HEADERS { 'If-Match': etag }

  If success (200/204):

    varControlNumber = concat(varOppId, '.', varYY, '.', formatNumber(varNext, '0000'))

    Update row gt_graphicrequest id=trigger.gt_requestid with { gt_graphiccontrolnumber: varControlNumber }

    Set varStamped = true

  Else if 412 Precondition Failed:

    Increment varRetry

    Delay rand(200ms .. 1000ms * pow(2, varRetry))

  Else: Terminate (Failed)

 

If not varStamped:

  Update row gt_graphicrequest with { gt_graphiccontrolnumber: 'PENDING' } and log to run history.
