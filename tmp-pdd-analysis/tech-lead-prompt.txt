- Name: Graphic Autonumber
- Description: •	Graphics Tracker is the Primary tool for Pursuit Services graphics support on Word response documents which is used for submitting and tracking progress of graphic requests, and inventory management for future reference/reuse. It is also used for storage of completed graphic files, associated templates, and production materials

•	Review RFX/project scope, design & submission requirements
•	Consult with deal team to gather and assess client insights and win themes
•	Discuss and brainstorm with deal team for materials to support submission/meeting

•	Leverage repository of reusable graphics/assets to create custom graphics
•	Use the Graphics tracker to track status for word only
•	Create graphics tracker to track status of individual graphics files
•	Update graphics tracker where graphics modifications requested - new slide within the file is created for each modification
•	Paste completed graphics files to Word file
•	Use iterative review cycle to refine graphics based on input from deal team
•	Format final documents and conduct quality check
•	Prepare files for final print production or digital submission
•	Confirm all final files have been uploaded for archiving/harvesting

•	All Graphic must be modified in some way to be incorporated into the proposal response and must be logged for tracking purposes.

Introduction

Overview and Context

Pursuit Services Graphic Designers (GDs) create engaging visual concepts and supporting graphics to illustrate key sales messages, data, and differentiators in print and digital pursuit deliverables with a consistent and compelling style.

 

Purpose



The scope of this automation is to auto generate the Graphic Control number. 



Process Overview

Graphics Tracker is the Primary tool for Pursuit Services graphics support on Word response documents which is used for submitting and tracking progress of graphic requests, and inventory management for future reference/reuse. It is also used for storage of completed graphic files, associated templates, and production materials

Pre-Process Steps:



The Graphic tracker will be enabled on individual teams’ channels for the deal supported by the Graphic Designer. 

Multiple designers can be supporting one deal and hence will be leveraging this tracker which is placed in the common team’s folder. 

All Graphic must be modified in some way to be incorporated into the proposal response and must be logged for tracking purposes.

Consecutive Graphic control number should be generated even if multiple designers are creating it, this should be based on the last created number. 

Post-Process steps



Graphic control number is created when the submitter or the graphic designer creates an entry on the Graphic Tracker which will help reference and identify graphics created by Pursuit Services

Process Steps 



Draft Graphic request is submitted on the tracker.

All Graphic must be modified in some way to be incorporated into the proposal response and must be logged for tracking purposes.

A notification can be sent to the designer that a new item has been added to the tracker. 

The designer updates the tracker that the Graphic is In Progress.

Designer notifies the submitter that this is now completed, reviewer will review and update status of Graphic Tracker Request Item if any edits or changes needed

Graphic tracker is updated to complete status once received confirmation from requestor that this can be incorporated into the proposal response and no further edits are required.
