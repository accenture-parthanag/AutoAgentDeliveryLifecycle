Bot Sizing & Capacity Planning
Bot Count▾
Bot Capacity Formula
Bots Required = ⌈ ( Daily Volume × AHTmins ) ÷ ( Processing Windowmins × Utilisation% ) ⌉ × Peak Factor
AHT = Average bot handling time per transaction (minutes) Processing Window = Available hours × 60 Utilisation = Target bot utilisation (typically 70–80%) Peak Factor = Peak Vol ÷ Avg Vol (add 20–30% buffer) ⌈ ⌉ = Ceiling — always round up
